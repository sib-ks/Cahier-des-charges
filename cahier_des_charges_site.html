Description utilisateur

Application de chat instantané permettant de discuter en temps réel via un serveur centralisé. Chaque utilisateur choisit un pseudo et peut envoyer/recevoir des messages dans une interface conviviale.

Fonctionnalités prévues

  • Création de compte avec pseudo
  • Connexion / déconnexion au serveur
  • Envoi et réception de messages
  • Affichage des pseudos devant chaque message
  • Historique de discussion visible
  • Confirmation avant fermeture
  • Options futures : messages privés, emojis, fichiers joints
